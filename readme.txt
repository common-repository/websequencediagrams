Add shortcode for WebSequenceDiagram.

== Description ==

This plugin will add button to TinyMCE Visual Editor which when clicked will append shortcode for WebSequenceDiagrams.

== Installation ==

1. Upload `wsd` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

**Features**
